6. Best Practices

  • Always include descriptive alt attributes for accessibility
  • Use relative paths for portability
  • Specify width/height to prevent layout shifts
  • Optimize images before adding to the library
  • Use appropriate formats (SVG for icons, PNG for graphics, JPG for photos)
